Módulo 7: Uso de Herramientas Externas

7.1 phpMyAdmin

Instalación y configuración:

Aprenderás a instalar y configurar phpMyAdmin, una herramienta web de código abierto utilizada para administrar bases de datos MySQL de manera visual.

# Instalación de phpMyAdmin en sistemas Linux (Debian/Ubuntu)
sudo apt-get update
sudo apt-get install phpmyadmin

# Configuración para conexión a servidor MySQL
sudo dpkg-reconfigure phpmyadmin

Uso de phpMyAdmin para administrar bases de datos:

Explorarás las funcionalidades de phpMyAdmin, como la administración de bases de datos, la creación y modificación de tablas, la ejecución de consultas SQL y la importación/exportación de datos.

Ejemplo: Creación de una nueva tabla con phpMyAdmin:

  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos deseada.
  3. Navega a la pestaña "Estructura".
  4. Haz clic en "Añadir tabla".
  5. Define las columnas y tipos de datos.
  6. Establece claves primarias y otros atributos.
  7. Haz clic en "Guardar".

7.2 Otras Herramientas de Administración

Exploración de herramientas externas para administración y visualización de datos en MySQL:

Además de phpMyAdmin, existen otras herramientas de administración y visualización de bases de datos MySQL, como MySQL Workbench y HeidiSQL. Aprenderás a explorar y utilizar estas herramientas para tareas de administración avanzadas.
