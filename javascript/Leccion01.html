Lección 1: Introducción a JavaScript

¿Qué es JavaScript y por qué es importante?

JavaScript es un lenguaje de programación utilizado principalmente para agregar interactividad y dinamismo a las páginas web. Es ampliamente compatible con los navegadores y permite crear experiencias más atractivas para los usuarios.

Configuración del entorno de desarrollo

Para comenzar a programar en JavaScript, solo necesitas un navegador web y un editor de texto. Puedes usar cualquier editor de texto como Notepad++ o Visual Studio Code.

Primer programa en JavaScript: Hola Mundo

Veamos cómo imprimir "Hola Mundo" en la consola del navegador utilizando JavaScript.


        <script>
          console.log("Hola Mundo");
        </script>

Comentarios y sintaxis básica

Los comentarios se utilizan para hacer anotaciones en el código sin afectar su funcionamiento. En JavaScript, los comentarios de una línea se crean con //, y los comentarios de varias líneas se crean entre /* y */.


        <script>
          // Esto es un comentario de una línea
          /* Esto es un comentario
          de varias líneas */
        </script>
