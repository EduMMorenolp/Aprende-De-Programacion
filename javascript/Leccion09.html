Lección 9: Trabajo con Fechas y Tiempos

Creación de objetos de fecha

JavaScript proporciona un objeto Date para trabajar con fechas y tiempos.


        <script>
          var fechaActual = new Date();
          console.log(fechaActual);

          var fechaEspecifica = new Date("2023-09-01T12:00:00");
          console.log(fechaEspecifica);
        </script>

Métodos de fecha y tiempo

El objeto Date tiene varios métodos para obtener información específica de las fechas y tiempos.


        <script>
          var fecha = new Date();

          var anio = fecha.getFullYear();
          var mes = fecha.getMonth();
          var dia = fecha.getDate();

          console.log(anio, mes, dia);
        </script>

Formato y manipulación de fechas

Es posible formatear y manipular fechas utilizando métodos específicos.


        <script>
          var fecha = new Date();
          var opciones = { year: 'numeric', month: 'long', day: 'numeric' };

          var fechaFormateada = fecha.toLocaleDateString("es-ES", opciones);
          console.log(fechaFormateada);

          fecha.setDate(fecha.getDate() + 7);
          console.log(fecha);
        </script>
