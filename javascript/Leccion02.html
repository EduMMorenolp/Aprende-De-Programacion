Lección 2: Variables y Tipos de Datos

Declaración de variables y asignación

En JavaScript, las variables se utilizan para almacenar y manipular datos. Se declaran con las palabras clave var, let o const.


        <script>
          // Declaración de variables
          var edad;
          let nombre;

          // Asignación de valores
          edad = 25;
          nombre = "Juan";

          console.log(nombre + " tiene " + edad + " años.");
        </script>

Tipos de datos

En JavaScript, existen varios tipos de datos, incluyendo números, cadenas, booleanos, undefined y null.


        <script>
          var numero = 42; // número entero
          var decimal = 3.14; // número decimal
          var nombre = "María"; // cadena de texto
          var esMayor = true; // valor booleano
          var noDefinido = undefined; // valor no definido
          var nulo = null; // valor nulo

          console.log(typeof numero);
          console.log(typeof nombre);
          console.log(typeof esMayor);
        </script>

Conversión entre tipos de datos

Es posible convertir entre diferentes tipos de datos utilizando funciones como parseInt y parseFloat.


        <script>
          var cadenaNumero = "25";
          var numero = parseInt(cadenaNumero);
          
          console.log(numero + 5); // 30
        </script>

Operadores básicos

Los operadores se utilizan para realizar operaciones matemáticas y de comparación.


        <script>
          var suma = 10 + 5;
          var resta = 20 - 8;
          var multiplicacion = 3 * 4;
          var division = 15 / 3;
          var modulo = 7 % 2; // resto de la división

          console.log(suma, resta, multiplicacion, division, modulo);
        </script>
