Lección 5: Arreglos

Creación y manipulación de arreglos

Los arreglos son colecciones ordenadas de elementos. Se declaran utilizando corchetes [].


        <script>
          // Creación de un arreglo
          var frutas = ["manzana", "banana", "naranja"];

          // Acceso a elementos por índice
          console.log(frutas[0]); // manzana
          console.log(frutas[1]); // banana
        </script>

Métodos de arreglo: push, pop, shift, unshift, slice

Los métodos de arreglo permiten agregar, eliminar y manipular elementos en un arreglo.


        <script>
          var numeros = [1, 2, 3];

          // Agregar elemento al final
          numeros.push(4);

          // Eliminar último elemento
          var ultimo = numeros.pop();

          // Agregar elemento al inicio
          numeros.unshift(0);

          // Eliminar primer elemento
          var primero = numeros.shift();

          // Copiar parte del arreglo
          var subarreglo = numeros.slice(1, 3);

          console.log(numeros);
        </script>

Iteración en arreglos con loops y forEach

Los bucles y el método forEach permiten recorrer y realizar operaciones en cada elemento de un arreglo.


        <script>
          var colores = ["rojo", "verde", "azul"];

          // Bucle for
          for (var i = 0; i < colores.length; i++) {
            console.log(colores[i]);
          }

          // forEach
          colores.forEach(function(color) {
            console.log(color);
          });
        </script>
