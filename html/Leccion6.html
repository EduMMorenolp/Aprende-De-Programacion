Lección 6: Tablas y Marcos en HTML

Etiqueta de tabla (<table>)

La etiqueta <table> se utiliza para crear tablas en una página web. Las tablas contienen filas (<tr>) y celdas (<td>) para organizar datos:


        <table>
          <tr>
            <td>Celda 1</td>
            <td>Celda 2</td>
          </tr>
          <tr>
            <td>Celda 3</td>
            <td>Celda 4</td>
          </tr>
        </table>

Encabezados de tabla (<th>)

La etiqueta <th> se utiliza para crear encabezados de columna o fila en una tabla. Ayuda a mejorar la accesibilidad y la estructura de la tabla:


        <table>
          <tr>
            <th>Nombre</th>
            <th>Edad</th>
          </tr>
          <tr>
            <td>Juan</td>
            <td>25</td>
          </tr>
        </table>

Etiqueta de marco (<iframe>)

La etiqueta <iframe> se utiliza para incrustar contenido de otras páginas web dentro de tu sitio. Es útil para mostrar mapas, videos y más:


        <iframe src="https://www.youtube.com/embed/VIDEO_ID"></iframe>
