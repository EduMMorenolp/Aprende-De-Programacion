Lección 1: Introducción a HTML

¿Qué es HTML y su importancia?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para crear la estructura y el contenido de las páginas web. Es la base de la web y se utiliza para definir la disposición de los elementos en una página, como encabezados, párrafos, imágenes y enlaces.

Estructura básica de un documento HTML

Etiquetas, elementos y atributos

HTML utiliza etiquetas para definir elementos y estructura. Las etiquetas suelen tener atributos que proporcionan información adicional. Por ejemplo:


        <p>Este es un párrafo de ejemplo.</p>
        <img src="imagen.jpg" alt="Descripción de la imagen">
        <a href="https://www.ejemplo.com">Enlace de ejemplo</a>

En el ejemplo, `

` define un párrafo, `` inserta una imagen y `` crea un enlace. Los atributos como `src`, `alt` y `href` proporcionan información específica para cada elemento.
