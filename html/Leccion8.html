Introducción a los Enlaces y Navegación

Los enlaces permiten conectar diferentes páginas web y recursos. Son fundamentales para la navegación entre páginas y sitios web. Se definen con la etiqueta <a>:


        <a href="https://www.ejemplo.com">Enlace a Ejemplo</a>

Tipos de Enlaces

  • Enlace a una página web externa: <a href="https://www.ejemplo.com">Enlace a Ejemplo</a>
  • Enlace a una página interna: <a href="pagina.html">Enlace a Página</a>
  • Enlace a un correo electrónico: <a href="mailto:correo@ejemplo.com">Enviar correo</a>
  • Enlace a un archivo: <a href="archivo.pdf">Descargar PDF</a>

Navegación en Menús

Los menús de navegación son comunes en sitios web. Pueden crearse utilizando listas no ordenadas y estilizándolos con CSS:


        <nav>
          <ul>
            <li><a href="inicio.html">Inicio</a></li>
            <li><a href="acerca.html">Acerca de</a></li>
            <li><a href="servicios.html">Servicios</a></li>
            <li><a href="contacto.html">Contacto</a></li>
          </ul>
        </nav>
