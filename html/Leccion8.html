Lección 8: CSS Básico

Introducción a CSS y su importancia:

CSS (Cascading Style Sheets) es un lenguaje utilizado para definir el estilo y la presentación de los elementos en una página web. Aprenderás sobre la importancia de CSS en la separación de la estructura HTML y la apariencia visual, lo que permite un diseño más flexible y mantenible.

Selectores y propiedades CSS:

Los selectores son patrones que se utilizan para seleccionar los elementos HTML a los que se aplicarán los estilos. Las propiedades definen cómo se verán los elementos seleccionados.

/* Ejemplo de selección de elementos y propiedades CSS */
h1 {
color: blue; /* Cambiar el color del texto a azul */
font-size: 24px; /* Cambiar el tamaño de fuente */
}

p {
font-family: Arial, sans-serif; /* Cambiar la fuente del párrafo */
}

Estilos en línea (style) y hojas de estilo externas ( ):

Aprenderás las dos formas principales de aplicar estilos CSS a tus documentos HTML: mediante estilos en línea y mediante hojas de estilo externas.

<!-- Estilo en línea -->
<h2 style="color: green;">Este es un encabezado con estilo en línea</h2>

<!-- Hoja de estilo externa -->
<head>
<link rel="stylesheet" type="text/css" href="estilos.css">
</head>

Sugerencias adicionales:

  • Organización: Mantén tus estilos organizados y utiliza comentarios para explicar las partes clave de tu hoja de estilo.
  • Consistencia: Mantén una consistencia en la aplicación de estilos en toda tu página web para lograr un aspecto coherente.
  • Prioridad: Familiarízate con la cascada y la especificidad en CSS para comprender cómo se aplican los estilos en diferentes situaciones.
