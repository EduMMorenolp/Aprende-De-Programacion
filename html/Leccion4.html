Lección 4: Imágenes y Multimedia en HTML

Etiqueta de imagen (<img>)

La etiqueta <img> se utiliza para mostrar imágenes en una página web. El atributo src especifica la ruta de la imagen y el atributo alt proporciona un texto alternativo para accesibilidad:


        <img src="imagen.jpg" alt="Descripción de la imagen">

Etiqueta de video (<video>)

La etiqueta <video> se utiliza para insertar videos en una página web. Puede contener múltiples fuentes de video para diferentes formatos y navegadores:


        <video controls width="400">
          <source src="video.mp4" type="video/mp4">
          <source src="video.webm" type="video/webm">
          Tu navegador no admite el elemento de video.
        </video>

Etiqueta de audio (<audio>)

La etiqueta <audio> se utiliza para reproducir archivos de audio. Al igual que con el elemento de video, puedes proporcionar múltiples fuentes de audio:


        <audio controls>
          <source src="audio.mp3" type="audio/mpeg">
          <source src="audio.ogg" type="audio/ogg">
          Tu navegador no admite el elemento de audio.
        </audio>
