Lección 4: Imágenes y Multimedia

Inserción de imágenes ():

Aprenderás cómo insertar imágenes en tus páginas web utilizando la etiqueta <img>. Las imágenes son una parte esencial para mejorar la presentación y el atractivo visual de tus sitios web.


<img src="imagen.jpg" alt="Descripción de la imagen" width="300">

Atributos src, alt y width:

  • src: El atributo src se utiliza para especificar la ruta de la imagen que deseas mostrar en la página.
  • alt: El atributo alt proporciona un texto alternativo para la imagen, que se muestra cuando la imagen no se puede cargar o para accesibilidad.
  • width: El atributo width establece el ancho de la imagen en píxeles.

Incorporación de audio y video (<audio>, <video>):

Aprenderás cómo incorporar archivos de audio y video en tus páginas web utilizando las etiquetas <audio> y <video>. Estas etiquetas permiten a los visitantes reproducir contenido multimedia directamente en tu sitio.
