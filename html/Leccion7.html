Lección 7: Estructura Semántica

Introducción a CSS

CSS (Cascading Style Sheets) se utiliza para aplicar estilos y diseño a las páginas web. Permite controlar la apariencia de los elementos HTML, como colores, fuentes, márgenes y más.

Roles en la estructura de una página web:

Aprenderás cómo las etiquetas semánticas juegan un papel fundamental en la estructura de una página web, ayudando a los motores de búsqueda y tecnologías de asistencia a comprender la relación y el significado de diferentes secciones del contenido.

Sugerencias adicionales:

  • Significado claro: Utiliza las etiquetas semánticas apropiadas para brindar un significado claro y estructurado a tu contenido.
  • Accesibilidad: Al proporcionar una estructura semántica adecuada, mejora la accesibilidad para usuarios con discapacidades visuales o tecnologías de asistencia.
  • Cascada de estilos: Combina etiquetas semánticas con estilos CSS adecuados para lograr un diseño atractivo y bien organizado.

Estilos externos

Los estilos externos se almacenan en un archivo CSS separado y se vinculan a la página HTML utilizando la etiqueta <link> en el encabezado:


        <link rel="stylesheet" href="estilos.css">
