Lección 7: Estilos y CSS en HTML

Introducción a CSS

CSS (Cascading Style Sheets) se utiliza para aplicar estilos y diseño a las páginas web. Permite controlar la apariencia de los elementos HTML, como colores, fuentes, márgenes y más.

Estilos en línea

Los estilos en línea se aplican directamente a elementos HTML utilizando el atributo style. Por ejemplo:


        <p style="color: blue; font-size: 18px;">Texto con estilo en línea</p>

Estilos internos

Los estilos internos se definen en la sección <style> dentro del encabezado HTML. Por ejemplo:


        <style>
          p {
            color: green;
          }
        </style>

Estilos externos

Los estilos externos se almacenan en un archivo CSS separado y se vinculan a la página HTML utilizando la etiqueta <link> en el encabezado:


        <link rel="stylesheet" href="estilos.css">
