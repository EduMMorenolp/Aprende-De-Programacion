Lección 9: Imágenes y Multimedia en HTML

Etiqueta de imagen (<img>)

La etiqueta <img> se utiliza para insertar imágenes en una página web. Se debe proporcionar la ubicación de la imagen en el atributo src y un texto alternativo en el atributo alt:


        <img src="imagen.jpg" alt="Descripción de la imagen">

Etiqueta de video (<video>)

La etiqueta <video> se utiliza para insertar videos en una página web. Se puede especificar varios formatos de video para garantizar la compatibilidad en diferentes navegadores:


        <video controls width="640" height="360">
          <source src="video.mp4" type="video/mp4">
          <source src="video.webm" type="video/webm">
          Tu navegador no admite el elemento <video>.
        </video>

Etiqueta de audio (<audio>)

La etiqueta <audio> se utiliza para insertar archivos de audio en una página web. Al igual que con los videos, se pueden especificar varios formatos para mejorar la compatibilidad:


        <audio controls>
          <source src="audio.mp3" type="audio/mpeg">
          <source src="audio.ogg" type="audio/ogg">
          Tu navegador no admite el elemento <audio>.
        </audio>
