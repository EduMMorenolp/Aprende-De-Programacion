Lección 5: Formularios en HTML

Etiqueta de formulario (<form>)

La etiqueta <form> se utiliza para crear un formulario en una página web. Puede contener varios elementos de entrada, como campos de texto, casillas de verificación y botones de envío:


        <form action="/procesar-formulario" method="post">
          
        </form>

Campos de texto (<input type="text">)

La etiqueta <input> con el atributo type="text" se utiliza para crear campos de entrada de texto:


        <label for="nombre">Nombre:</label>
        <input type="text" id="nombre" name="nombre">

Casillas de verificación (<input type="checkbox">)

La etiqueta <input> con el atributo type="checkbox" se utiliza para crear casillas de verificación:


        <label>
          <input type="checkbox" name="suscripcion" value="si"> Suscribirse al boletín
        </label>

Botones de envío (<input type="submit">)

La etiqueta <input> con el atributo type="submit" se utiliza para crear botones de envío de formulario:


        <input type="submit" value="Enviar">
