Lección 2: Etiquetas y Elementos Básicos en HTML

Etiquetas de encabezado: <h1> a <h6>

Las etiquetas de encabezado se utilizan para mostrar títulos y subtítulos en una página web. Hay seis niveles disponibles, donde <h1> es el nivel más alto y <h6> el más bajo:


        <h1>Título de nivel 1</h1>
        <h2>Título de nivel 2</h2>
        <h3>Título de nivel 3</h3>
        <h4>Título de nivel 4</h4>
        <h5>Título de nivel 5</h5>
        <h6>Título de nivel 6</h6>

Párrafos: <p> y <br>

La etiqueta <p> se utiliza para crear párrafos de texto. La etiqueta <br> se utiliza para insertar un salto de línea dentro de un párrafo:


        <p>Este es un párrafo.</p>
        <p>Este es otro párrafo.<br>Aquí hay una línea nueva.</p>

Etiquetas de formato: <strong>, <em>, <u>, <del>, <sup>, <sub>

Estas etiquetas se utilizan para aplicar formato y significado especial al texto:


        <p>Texto <strong>importante</strong>.</p>
        <p>Texto <em>enfatizado</em>.</p>
        <p>Texto <u>subrayado</u>.</p>
        <p>Texto <del>borrado</del>.</p>
        <p>10<sup>2</sup> es igual a 100.</p>
        <p>H<sub>2</sub>O es agua.</p>

Comentarios en HTML

Los comentarios se utilizan para agregar notas explicativas en el código que no se muestran en la página web:


        <!-- Esto es un comentario. -->
