Lección 10: Formularios en HTML

Introducción a los Formularios

Los formularios permiten a los usuarios enviar datos a un servidor web. Son esenciales para la recopilación de información, como registros, comentarios y más. Se definen utilizando la etiqueta <form>:


        <form action="procesar.php" method="post">
          
        </form>

Campos de Entrada

Los campos de entrada permiten a los usuarios ingresar datos. Algunos tipos comunes son:

  • Texto: <input type="text" name="nombre">
  • Contraseña: <input type="password" name="contrasena">
  • Correo electrónico: <input type="email" name="correo">
  • Número: <input type="number" name="edad">
  • Checkbox: <input type="checkbox" name="acepto">
  • Radio: <input type="radio" name="genero" value="masculino"> Masculino
  • Área de texto: <textarea name="comentarios"></textarea>

Etiqueta de Selección (<select>)

La etiqueta <select> se utiliza para crear listas desplegables. Cada opción se define con la etiqueta <option>:


        <select name="pais">
          <option value="usa">Estados Unidos</option>
          <option value="canada">Canadá</option>
          <option value="mexico">México</option>
        </select>

Botones de Envío (<input type="submit">)

El botón de envío permite enviar los datos del formulario al servidor. Se define con la etiqueta <input> y el atributo type="submit":


        <input type="submit" value="Enviar">
