Lección 2: Sintaxis y Variables

Sintaxis Básica de Java

La sintaxis en Java se refiere a la estructura y reglas que debes seguir al escribir código. Aquí hay un ejemplo de una declaración de clase en Java:

        
public class MiClase {
    // Código de la clase
}

Declaración y Tipos de Variables

Las variables son contenedores que almacenan datos en programas Java. Puedes declarar variables de diferentes tipos, como:

        
int edad = 25;
double altura = 1.75;
char inicial = 'J';
String nombre = "Juan";

Operadores y Expresiones

Los operadores se utilizan para realizar operaciones en variables y valores. Algunos ejemplos de operadores son: 

        
int suma = 10 + 5;
double division = 20.0 / 4;
boolean esMayor = edad > 18;

Entrada y Salida de Datos

Java te permite interactuar con el usuario utilizando la entrada y salida de datos. Aquí hay un ejemplo de cómo solicitar y mostrar datos:

        
import java.util.Scanner;

public class EntradaSalida {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Ingresa tu nombre: ");
        String nombre = scanner.nextLine();
        System.out.println("Hola, " + nombre + "!");
    }
}

Con esto, has completado la Lección 2 y tienes una comprensión sólida de la sintaxis de Java, cómo trabajar con variables, utilizar operadores y realizar entrada y salida de datos.
